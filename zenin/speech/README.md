# speech

Responsabilidad: preparar contratos para voz futura.

ZENIN podria escuchar o hablar en una fase posterior, pero esta carpeta no activa microfono, no reproduce audio y no solicita permisos.

Relaciones:

- `conversation` podra usar transcripciones futuras;
- `ui` podra mostrar estados de voz;
- `models` podria recibir texto sintetizado, no audio crudo.
