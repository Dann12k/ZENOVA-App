# observation-engine

Responsabilidad: detectar patrones sin IA y sin diagnosticar.

Observa:

- horario habitual;
- herramienta favorita;
- dia con mas estres;
- dia con mejor animo;
- tiempo promedio;
- herramientas abandonadas;
- habitos frecuentes;
- horas de sueno;
- tendencias.

Relaciones:

- recibe eventos desde `brain`;
- alimenta `emotion-memory`, `care-engine` y `daily-routine`;
- no genera mensajes.
