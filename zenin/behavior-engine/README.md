# behavior-engine

Responsabilidad: decidir que hace ZENIN.

No redacta respuestas. Devuelve acciones de comportamiento como celebrar, motivar, acompanar, sugerir o dar bienvenida especial.

Relaciones:

- consume emocion, cuidado, presencia, relacion y eventos;
- entrega acciones a `scheduler`;
- puede activar `celebration-engine` o `notification-engine`.
