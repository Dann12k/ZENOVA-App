# recommendations

Responsabilidad: sugerir herramientas, rutinas y recursos con logica propia de ZENOVA.

La IA futura puede explicar o personalizar, pero no debe decidirlo todo. Este modulo debe poder funcionar sin proveedor externo.

Relaciones:

- recibe contexto desde `context`;
- puede leer preferencias desde `memory`;
- entrega candidatos a `decision-engine` y a futuras superficies de UI.
