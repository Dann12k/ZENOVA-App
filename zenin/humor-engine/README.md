# humor-engine

Responsabilidad: ofrecer humor ligero, ocasional y respetuoso.

No debe ser ofensivo, infantil ni insistente. Usa plantillas combinables para producir cientos de variaciones sin depender de IA y registra lo usado para evitar repeticion constante.

Relaciones:

- recibe estilo desde `personality-engine`;
- puede alimentar `celebration-engine`;
- nunca se usa en contextos de alto estres o cuidado sensible.
