# relationship-engine

Responsabilidad: modelar la relacion entre ZENIN y el usuario con cambios lentos.

Variables:

- confianza;
- frecuencia;
- constancia;
- progreso;
- interaccion;
- afinidad.

Relaciones:

- recibe eventos desde `brain`;
- alimenta `personality-engine`, `presence-engine` y `care-engine`;
- no guarda datos sensibles ni reemplaza memoria.
