# celebration-engine

Responsabilidad: preparar celebraciones saludables sin gamificacion excesiva.

Puede definir confeti, particulas, brillos, sonidos preparados, medallas pequenas y mensajes unicos. No renderiza la celebracion.

Relaciones:

- recibe logros desde `achievement-system`;
- puede usar frases desde `humor-engine`;
- entrega un modelo visual a `notification-engine` o futura UI.
